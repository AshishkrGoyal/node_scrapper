Send Messages To Your Friends

This tool allows you to send messages to your facebook friends.

Write message that you want to send to your selected friends
Maximum character limit for text messages is 240 characters.
Important information
Use this tool with caution !!
There is a limit to number of requests you can send to Facebook API servers.
Don't send messages to more than 30 friends at a time.
Don't flood Facebook servers or your account may get banned or deactivated.
Select time interval between requests
Select friends to whom you want to send a message.
