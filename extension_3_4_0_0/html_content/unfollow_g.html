Unfollow multiple Facebook groups

This tool will allow you to unfollow multiple Facebook groups at once.
Select list of groups that you want to unfollow.
After that click on submit button.

Important information
Use this tool with caution !!
There is a limit to number of requests you can send to Facebook API servers.
Don't unfollow too many Facebook groups at a time.
Don't flood Facebook servers or your account may get banned or deactivated.
Select time interval between requests
Select groups to unfollow.
